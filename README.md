## Usage

```js
var seeditCityPc = require('seedit-city-pc');
// use seeditCityPc
```

## API

### getValue

获取值，返回数组：

```json
[{
	id: 2,
	name: "天津市"
}, {
	id: 56,
	name: "河东区"
}]
```

## Events

### change

监听变化，事件参数和`getValue`一致
